# 1.3-Manipulaci-n-y-agrupaci-n-de-datos-con-SQL
Resolver los problemas mencionados generando las consultas, con la base de datos WideWordImporters.

Solución de Consultas en SQL Server - WideWorldImporters

Este repositorio contiene la solución a los ejercicios prácticos de bases de datos utilizando la base de datos de ejemplo **WideWorldImporters** en SQL Server Management Studio (SSMS). 

El objetivo de esta actividad es aplicar combinaciones de tablas (`JOIN`) y comprender el flujo lógico de la información y la estructura relacional de los datos.
